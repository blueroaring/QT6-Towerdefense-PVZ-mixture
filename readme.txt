本程序不包含任何绝对路径,请放心放在任何合理位置
只有音乐是必须和build在同一个层级才能放歌的，要不听不到声音
本项目使用QT6编写，采用了QT6新加的库，因此请在QT6版本的QTCreateor打开Towerdefense.pro文件执行（如果执行.exe,可以运行，但是没有声音）
如果你使用visual studio 或者vscode的qt插件，不保证能否运行，最好使用qt6原生环境
本文件里不含pro.user,打开.pro文件后先对其进行本地的build在运行!不要用MSVC编译！用mingGW(qt默认的)即可，这个是qmake文件
1.修改地图要修改JSON的数据，这里已经提供了level1-level4的地图数据（不会出怪），和存档save4（第四关），可以通过修改level1-level4的map中的值（如果改的有问题会报错，已做好错误检测，但是不保证全能检测到），改数就是对从左到右，从上到下的区块修改（不能玩），区块的意义详见存档界面
2.狂暴：伤害增加
弱化：群攻
治疗术：固定时间回血
晕眩：减速或定身，只有寒冰射手装备才会定身
斩杀：直接秒杀一个僵尸，但是秒杀后词条会消失（如果同时还有群攻可以秒杀一群僵尸）
治疗术，斩杀，弱化只能用于近战塔，晕眩只能用于远程塔，狂暴都能用
3.关于卡槽和词缀的加入、卸载：如果要种植物，先点击卡槽再点击地图就行，不能种的话种不下去的，词缀同理，不能安装的话安不上去的
卸载：植物用铲子铲就行，点击铲子，再点击植物；词缀的话，右键即可卸下（会优先卸载早的那个）
4.敌人词缀：当我方近战塔血量小于一定比例后，（使防御塔掉到临界值的）僵尸会获得一个词缀（最多一个），会显示在头顶，不会卸下
疾跑：跑得快
净化：免疫减速和定身
干扰：近战塔伤害无效（包括樱桃炸弹）
闪现：瞬间移动一个格的距离（然后词缀会消失）
5.存档的话，如果level=5,那么就是观赏模式，其实这不好，后面会改，如果level被改成0-4,会出Bug（贴图和实际不匹配）
6.关卡（1-4）执行过程中可以存档，存完也可以继续玩，退出也会问你是否要存档，会存到savei.json里(i是你的关卡数)
加载存档这个文件就能继续游戏，但是敌人会从最近的波次重新刷，也不会保留已安装的词缀（未来会改）
7.主界面第一个按钮是直接打第一关（暂时没写最高通关记录），第二个按钮是自己选关或者加载存档，第三个是教程，第四个是图鉴（没写），第五个是生成可修改的观赏地图（同样在加载存档处加载查看）
